Use solarized colorscheme in true color

2016/09/19 Mon

Follow @yukimemi

前回の記事(True color in iTerm2 Tmux Neovim | YUKIMEMI)で、 iterm と neovim でtrue colorを使う方法はわかったんだけど、なぜか、 solarized の colorscheme が有効にならなかった。

通常の solarized ではダメなようで、true color対応したやつを使えばいいみたい。

[[plugins]]
repo = 'lifepillar/vim-solarized8'

colorscheme solarized8_dark

これでできた。
